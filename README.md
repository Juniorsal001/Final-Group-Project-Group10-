# Supermarket Website — Group Project 10

A responsive supermarket website that lets customers browse products, view product details, and submit orders or contact requests. Built as a group assignment to demonstrate front-end web development using plain HTML, CSS, and JavaScript.

Features
- Responsive layout for desktop and mobile
- Product catalog and product pages
- Contact / order form
- Sign-in and VIP pages (UI only)

Technologies
- HTML, CSS, JavaScript (vanilla)

Getting Started
1. Clone the repository:

	git clone https://github.com/your-username/Final-Group-Project-Group10-.git

2. Open the project folder in your code editor or file explorer.
3. Launch the site by opening `Market.html` in your browser (this serves as the product catalog/home page). Other pages:

- `Form.html` — Contact / Order form
- `Sign in.html` — Sign-in UI
- `VIP.html` — VIP page UI
- `Product.html` — Product detail template

Project Structure
- Contact Us.html / Contact Us.css — Contact page and styles
- Market.html / Market.css / Market.js — Main product listing
- Product.html / Product.css / Product.js — Product detail view
- Form.html / Form.css / form.js — Contact / order form
- Sign in.html / Sign in.css / Sing in.js — Authentication UI
- VIP.html / VIP.css / VIP.js — VIP page
- Images Folder/ — Images used by the site
